# Target Pricing Opportunity

## Overview
Use market data, cost models, and best-in-class pricing to establish negotiation targets and achieve cost reductions through should-cost analysis.

## Context
The Target Pricing micro-agent evaluates proof points with the hypothesis that **price variances and market benchmarks reveal negotiation opportunities**.

## Proof Points (4)

| Code | Name | Context for Target Pricing |
|------|------|---------------------------|
| PP_PRICE_VARIANCE | Price Variance for Identical SKUs | High variance = use best price as target for all |
| PP_TARIFF_RATE | Tariff Rate | High differential = optimize sourcing mix |
| PP_COST_STRUCTURE | Cost Structure | Commodity-driven = index-based pricing possible |
| PP_UNIT_PRICE | Unit Price | Above benchmark = negotiation opportunity |

## Savings Benchmark
- **Low**: 1%
- **High**: 2%
- **Addressable Spend**: 80%

## Example Insights
- "Best-in-class price is 18% below average - use $1,245 as target for negotiations"
- "Tariff differential of 15-25% between Malaysia and China origins"
- "Cost structure is commodity-driven with raw materials at 65% - index-based pricing recommended"
- "Current unit price $1,520 is 12% above market benchmark - target price $1,350"

## Key Difference from Other Opportunities
Same data, different interpretation:
- **Volume Bundling**: Price variance = "consolidate and standardize pricing"
- **Target Pricing**: Price variance = "use lowest price as negotiation target"
